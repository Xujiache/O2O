## 一、项目概述

### 1\.1 项目背景

针对同城本地生活即时配送需求，搭建集**外卖点餐 \+ 同城跑腿**为一体的 O2O 服务平台，打通用户下单、商户接单出餐、骑手配送、平台运营管控、财务结算全业务闭环，覆盖同城餐饮外卖、物品帮送 / 帮取、商品帮买、便民帮办等核心场景，为 C 端用户、入驻商户、配送骑手提供全链路数字化服务。

### 1\.2 项目核心目标

1. 实现外卖 \+ 跑腿双业务线的一体化流程管控，支持多端业务协同

2. 搭建高可用、高并发的即时配送系统，保障订单全流程的稳定性与实时性

3. 提供完善的运营、财务、风控管理能力，支撑平台规模化商业运营

4. 保障多端用户体验，满足用户便捷下单、商户高效管店、骑手灵活接单的核心诉求

### 1\.3 业务范围

- 【在需求范围内】：外卖点餐全流程、同城跑腿全流程、多端核心业务功能、平台运营管理、财务分账结算、基础风控与权限管理

- 【需求边界外】：商户 ERP 深度集成、第三方外卖平台对接、供应链管理、跨境 / 跨城配送、金融信贷服务、线下门店收银系统

### 1\.4 核心角色定义

|角色名称|角色说明|
|---|---|
|C 端用户|微信小程序端使用者，外卖下单、跑腿服务发起方|
|入驻商户|外卖商家，负责商品管理、订单接单出餐、店铺运营|
|平台骑手|配送服务提供方，负责外卖订单、跑腿订单的取送履约|
|平台管理员|管理后台使用者，负责平台全业务线的运营、管控、审核、风控|

## 二、核心业务主流程

### 2\.1 外卖业务主流程

用户下单→用户支付→商户接单→商户出餐→骑手接单→骑手取餐→骑手配送→用户确认收货→订单完成→平台与商户 / 骑手分账结算

### 2\.2 跑腿业务主流程

用户发起跑腿需求→系统价格预估→用户下单支付→系统派单 / 骑手抢单→骑手取件 / 完成帮买→骑手配送→收件人 / 用户确认收货→订单完成→平台与骑手分账结算

## 三、功能性需求（分端详细说明）

### 3\.1 用户端（微信小程序，uni\-app 开发）

#### 3\.1\.1 首页模块

1. 定位服务：自动获取用户当前位置，支持手动切换收货地址，定位精度匹配配送范围校验

2. 搜索功能：支持商家、商品、跑腿服务关键词搜索，历史搜索记录、热门搜索推荐

3. 运营位：顶部 banner 轮播、活动弹窗、快捷入口（外卖、跑腿、优惠活动、客服）

4. 推荐模块：附近热门商家、优惠商品、销量排行、新店推荐，支持按距离、销量、评分筛选

5. 公告栏：平台运营公告、活动通知滚动展示

#### 3\.1\.2 外卖点餐模块

1. 商家列表：按配送范围、营业状态筛选，支持按距离、销量、评分、价格排序，展示商家配送费、起送价、配送时长、评分、优惠活动

2. 商家详情：店铺信息、营业时间、配送规则、评分评价、商品分类、商品列表、店铺公告、优惠活动

3. 商品管理：商品详情展示（规格、价格、库存、销量、评价）、规格选择、加入购物车、商品收藏

4. 购物车：商品增减、规格修改、清空、价格实时计算、凑单提示、下单结算入口

5. 下单结算：收货地址确认、配送时间选择、备注填写、优惠券抵扣、发票申请、订单金额明细、支付方式选择

6. 评价体系：订单完成后商家 / 商品评分、文字 / 图片评价、评价回复查看、商家评分统计

#### 3\.1\.3 同城跑腿模块

1. 服务分类：帮送、帮取、帮买、帮排队四大核心场景，支持自定义跑腿需求

2. 帮送 / 帮取功能：寄件 / 收件地址填写、物品类型 / 重量选择、保价服务、取件时间选择、备注填写、运费实时预估、下单支付

3. 帮买功能：购买地址选择、商品清单填写、预算金额、购买时间要求、备注填写、服务费 \+ 商品预估金额计算、下单支付

4. 帮排队功能：排队场所、排队类型、排队时长预估、备注填写、服务费计算、下单支付

5. 订单跟踪：跑腿订单实时状态更新、骑手定位实时展示、取 / 送件凭证上传查看、骑手联系方式一键拨打

#### 3\.1\.4 订单管理模块

1. 全量订单列表：外卖订单 \+ 跑腿订单统一管理，支持按订单状态、订单类型筛选

2. 订单状态分类：待支付、待接单、待取餐 / 取件、配送中、已完成、已取消、售后中

3. 订单操作：取消订单、申请退款 / 售后、确认收货、联系骑手 / 商家、查看订单详情、订单投诉、再次下单

4. 售后管理：退款 / 售后申请提交、凭证上传、售后进度跟踪、售后记录查询、平台仲裁申请

#### 3\.1\.5 个人中心模块

1. 账号管理：微信授权一键登录、个人信息编辑、头像设置、实名认证（跑腿大额订单必填）

2. 地址管理：收货地址新增、编辑、删除、设为默认，地址智能识别

3. 资产与优惠：我的钱包、余额明细、优惠券 / 红包、积分管理、邀请有礼

4. 收藏管理：商家收藏、商品收藏列表管理

5. 发票管理：发票抬头管理、开票申请、开票记录查询

6. 客服与帮助：在线客服、智能机器人、常见问题、意见反馈

7. 系统设置：消息通知开关、隐私设置、账号安全、关于我们

#### 3\.1\.6 支付与消息模块

1. 支付功能：微信支付为主，支持余额支付，支付状态实时同步、支付失败重试、支付超时自动取消

2. 消息通知：微信订阅消息推送（订单状态变更、活动通知、退款通知）、小程序内消息中心、订单状态语音提醒

### 3\.2 商户端（uni\-app 开发，支持编译为 APP / 微信小程序）

#### 3\.2\.1 登录与认证模块

1. 账号登录：手机号验证码登录、密码登录

2. 资质认证：商户入驻信息提交、营业执照、食品经营许可证、法人信息、店铺信息上传、入驻进度查询

3. 账号管理：子账号创建、权限分配、账号安全设置、密码修改

#### 3\.2\.2 店铺管理模块

1. 店铺信息：店铺名称、头像、营业时间、地址、配送范围、起送价、配送费、店铺公告编辑与更新

2. 营业状态：营业中 / 打烊状态一键切换，自动按营业时间启停营业，支持临时歇业设置

3. 配送设置：配送范围自定义、配送费阶梯定价、预订单设置、配送时长设置

4. 评价管理：用户评价查看、评价回复、差评申诉、评分统计分析

#### 3\.2\.3 订单管理模块

1. 订单实时提醒：新订单语音 \+ 弹窗提醒，支持订单自动接单设置

2. 订单全流程管理：待接单、待出餐、配送中、已完成、已取消、售后中订单分类管理

3. 订单操作：接单 / 拒单（需填写拒单原因）、出餐完成标记、打印订单、取消订单、退款审核、联系用户 / 骑手、查看订单详情

4. 异常订单处理：用户取消订单申请审核、退款申请审核、异常订单上报、平台仲裁配合

5. 订单打印：对接蓝牙打印机，支持订单小票自动打印、批量打印

#### 3\.2\.4 商品管理模块

1. 商品分类：分类新增、编辑、删除、排序、上下架

2. 商品管理：商品新增、编辑、删除、上下架，商品名称、图片、规格、价格、库存、配料、销量、标签设置

3. 规格管理：多规格多价格设置、库存独立管理

4. 套餐管理：套餐商品组合、套餐价格设置、套餐库存管理

5. 优惠商品：特价商品设置、限时折扣活动设置

#### 3\.2\.5 财务结算模块

1. 数据概览：今日 / 昨日 / 月度营业额、订单量、实收金额、平台分佣、退款金额统计

2. 账单明细：每笔订单的收入、支出、分佣、退款明细，支持按时间、订单号筛选

3. 提现管理：账户余额查看、提现申请、提现记录查询、提现银行卡管理

4. 发票管理：向平台申请开票、开票记录查询、发票资质管理

#### 3\.2\.6 数据统计与消息模块

1. 数据统计：订单量统计、营业额趋势、商品销量排行、用户复购率、评分统计、营业数据分析

2. 消息中心：订单通知、退款通知、平台公告、运营活动通知、违规提醒

3. 营销工具：店铺优惠券设置、满减活动、折扣活动、新品推荐、拼单活动配置

### 3\.3 骑手端（uni\-app 开发，优先编译为 iOS/Android APP）

#### 3\.3\.1 登录与认证模块

1. 账号登录：手机号验证码登录、密码登录

2. 资质认证：骑手实名认证、健康证上传、电动车信息、从业资格认证、入驻审核进度查询

3. 保证金管理：保证金缴纳、补缴、退还申请、保证金记录查询

#### 3\.3\.2 接单大厅模块

1. 订单列表：附近可接订单（外卖 \+ 跑腿）展示，按距离、配送费、顺路程度排序，展示订单详情、取送地址、配送费、取件时间要求

2. 接单模式：抢单模式、系统智能派单模式切换，派单规则设置（接单范围、订单类型偏好）

3. 订单操作：抢单、接单、转单申请、拒单（需填写原因）、查看订单详情

4. 顺路单匹配：系统自动推荐与当前配送订单顺路的订单，支持批量接单

#### 3\.3\.3 订单配送模块

1. 订单全流程管理：待取件、配送中、已完成、已取消订单分类管理

2. 核心配送操作：

    - 取件环节：一键导航到取件地址、联系用户 / 商家、取件码核验、取件完成标记、取件凭证上传

    - 配送环节：实时定位上报、一键导航到收件地址、联系收件人、送达确认、送达凭证上传、订单完成标记

3. 异常处理：联系不上用户 / 商家、商家出餐慢、物品破损、地址错误等异常情况上报、凭证上传、订单取消申请、平台客服求助

4. 导航服务：对接高德 / 百度地图 SDK，内置导航、路线规划、配送距离实时计算

#### 3\.3\.4 我的钱包模块

1. 数据概览：今日 / 昨日 / 月度配送单量、配送收入、奖励、罚款统计

2. 账单明细：每笔订单的收入、奖励、罚款、补贴明细，支持按时间、订单类型筛选

3. 提现管理：账户余额查看、提现申请、提现记录查询、提现银行卡管理

4. 薪资结算：配送费结算规则展示、结算周期说明、薪资明细导出

#### 3\.3\.5 考勤与统计模块

1. 考勤管理：上下班打卡、在线时长统计、排班查看、请假申请

2. 数据统计：配送单量趋势、配送准时率、好评率、差评记录、投诉记录统计

3. 等级与奖惩：骑手等级规则、积分管理、奖励记录、罚款记录、申诉入口

#### 3\.3\.6 消息与设置模块

1. 消息中心：订单推送通知、系统公告、奖惩通知、平台活动通知，订单状态语音播报

2. 系统设置：接单设置、消息通知开关、语音播报设置、账号安全、帮助中心、在线客服、意见反馈

3. 紧急求助：一键联系平台客服、紧急报警功能

### 3\.4 平台管理后台（Web 端）

#### 3\.4\.1 数据大盘模块

1. 核心数据概览：实时订单量、今日交易额、累计用户数、累计商户数、在线骑单数、今日配送完成率

#### 3\.4\.2 用户管理模块

1. 用户列表：全平台用户信息查询、编辑，支持按手机号、注册时间、订单量筛选

2. 用户详情：用户订单记录、消费数据、地址管理、优惠券、余额明细、投诉记录查看

3. 用户风控：用户账号封禁 / 解封、黑名单管理、异常消费行为监控、违规处理

#### 3\.4\.3 商户管理模块

1. 入驻审核：商户入驻申请列表、资质审核、信息核验、审核通过 / 驳回（需填写原因）

2. 商户列表：全平台商户信息管理，支持按营业状态、区域、入驻时间筛选，商户信息编辑、营业状态管控、账号封禁 / 解封

3. 店铺管理：店铺配送范围、营业状态、评分管理、违规处理、店铺公告审核

4. 商户风控：商户违规行为监控、差评处理、投诉仲裁、保证金管理

#### 3\.4\.4 骑手管理模块

1. 入驻审核：骑手入驻申请列表、实名认证、健康证资质审核、审核通过 / 驳回

2. 骑手列表：全平台骑手信息管理，支持按在线状态、接单区域、等级筛选，骑手信息编辑、账号封禁 / 解封

1. 全量订单管理：外卖 \+ 跑腿全平台订单查询，支持按订单号、手机号、订单状态、时间范围、订单类型筛选

1. 商品管理：全平台商品信息查询、上下架管控、违规商品处理、商品分类管理

2. 内容管理：首页 banner、运营位、公告、热门搜索、快捷入口配置与管理

3. 评价管理：全平台用户评价查询、违规评价处理、差评申诉审核

#### 3\.4\.7 运营管理模块

1. 优惠活动管理：优惠券 / 红包创建、发放规则配置、活动起止时间、使用范围设置、发放数据统计

2. 营销活动：满减活动、折扣活动、拼单活动、邀请有礼、新用户福利活动配置与管理

3. 消息推送：全平台用户 / 商户 / 骑手定向消息推送、订阅消息模板配置

#### 3\.4\.8 财务管理模块

1. 财务概览：平台营收、分佣支出、退款金额、流水统计、资金趋势图表

2. 分账管理：平台分佣比例配置、商户 / 骑手分账规则设置、自动分账管理、分账记录查询

3. 提现管理：商户 / 骑手提现申请审核、提现驳回、打款确认、提现记录查询

5. 发票管理：商户 / 用户开票申请审核、开票记录管理、发票资质管理

1. 权限管理：角色创建、权限分配、管理员账号管理、操作日志查询

2. 字典管理：平台基础数据配置、订单状态、违规类型、行业分类等字典维护
