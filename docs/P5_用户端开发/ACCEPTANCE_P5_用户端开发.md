## 二、非功能验收

| 项 | 标准 |
|---|---|
| 首屏加载 | ≤ 2s（强网）/ ≤ 3s（弱网） |
| 页面切换 | ≤ 300ms |
| 主包大小 | ≤ 2MB |
| 分包大小 | 每分包 ≤ 2MB |
| 状态推送延迟 | ≤ 3s |
| 微信基础库 | ≥ 2.0 兼容 |
| 机型兼容 | iPhone 12/13/14 + 主流 Android 无错乱 |
| 埋点 | ≥ 30 事件，成功率 ≥ 99% |

## 三、安全与合规
- 用户授权文案清晰，拒绝不影响浏览
- 敏感信息（mobile/身份证）脱敏展示
- 合规：无隐藏收集、无越界调用
