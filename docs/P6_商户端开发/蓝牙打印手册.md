# O2O 商户端 · 蓝牙打印机使用手册

> **目的**：为商户提供完整的蓝牙打印机连接、打印、故障排查指南  
> **支持机型**：芯烨 XP-58、佳博 GP-58MB、汉印 HM-A300 等市售 ESC/POS 兼容打印机  
> **对应**：T6.40 蓝牙打印机 SDK 封装

---

## 一、硬件准备

### 1.1 推荐机型

| 品牌型号 | 纸宽 | 接口 | 备注 |
|---|---|---|---|
| 芯烨 XP-58IIH | 58mm | 蓝牙 4.0 | 商户端默认 |
| 佳博 GP-58MB | 58mm | 蓝牙 4.0 | 备选 |
| 汉印 HM-A300 | 80mm | 蓝牙 5.0 | 大店推荐 |
| 思普瑞特 SP-RMD8 | 80mm | 蓝牙 4.2 | 备选 |

### 1.2 纸卷规格

- 58mm：57.5±0.5mm × Φ40mm 热敏纸
- 80mm：79.5±0.5mm × Φ80mm 热敏纸

## 二、首次连接（5 步）

1. 打开手机蓝牙
2. 给打印机装纸 + 通电（指示灯亮）
3. 商户 APP → 我的 → **打印机设置**
4. 点击「**扫描打印机**」按钮（8 秒扫描超时）
5. 选择列表中你的打印机（按 RSSI 信号强度排序，越靠前越近）

> 连接成功后，连接状态显示「已连接 ✓」

---

## 三、打印参数配置

| 参数 | 取值 | 说明 |
|---|---|---|
| 纸宽 | 58 / 80mm | 与硬件一致 |
| 自动打印 | 开/关 | 接单后自动打印（**强烈建议开启**） |
| 自动重连 | 开/关 | 蓝牙断开后自动尝试连接 |
| 每单份数 | 1-5 | 厨房联 / 配送联 / 客户联 多联打印 |

## 五、操作指引

### 5.1 重新打印

订单详情 → 操作栏「**重新打印**」按钮（仅 status in [20, 30, 40] 时可见）  
单击：发送至队列；自动重试 3 次

### 5.2 切换打印机

- **删除当前**：打印机设置 → 「断开连接」
- **添加新机**：打印机设置 → 「扫描打印机」→ 选新设备

### 5.3 多店一机

- 单台打印机可同时绑定多个店铺
- 切换店铺后，打印机仍连接
- 不同店铺订单按入队顺序串行打印

## 六、故障排查

| 现象 | 可能原因 | 解决 |
|---|---|---|
| 扫描不到设备 | 蓝牙未开 / 打印机未通电 | 检查蓝牙开关 + 打印机电源 |
| 连接失败 | 距离过远 / 已被其他设备连接 | 靠近打印机 / 重启打印机 |
| 打印乱码 | 编码不一致 | 检查打印机是否支持 UTF-8 |
| 走纸但不打印 | 纸卷反向 / 不是热敏纸 | 翻转纸卷 / 更换标准热敏纸 |
| 部分行打印不全 | 纸宽不一致（设了 80 实际 58） | 调整「打印参数 → 纸宽」 |
| 一单打印多次 | 自动打印 + 手动打印重复 | 关闭自动打印 |
| 卡纸 | 纸卷过厚 / 纸路异物 | 取出纸卷重装 |
| 蓝牙频繁断开 | 距离过远 / 多设备干扰 | 缩短距离 / 关掉其他蓝牙设备 |

## 七、技术细节（开发参考）

### 7.1 ESC/POS 指令封装位置

`商户端/src/utils/escpos.ts` —— `EscPosBuilder` 类

### 7.2 主要指令

| 指令 | 字节 | 作用 |
|---|---|---|
| `ESC @` | 0x1B 0x40 | 初始化 |
| `ESC a n` | 0x1B 0x61 n | 对齐（0 左 / 1 中 / 2 右） |
| `GS ! n` | 0x1D 0x21 n | 字号（0 正常 / 0x11 双倍 / 0x22 三倍） |
| `GS V 1` | 0x1D 0x56 0x01 | 全切 |
| `GS k 73` | 0x1D 0x6B 0x49 | CODE128 一维码 |
| `ESC B n d` | 0x1B 0x42 n d | 蜂鸣（n 次 d 秒） |

### 7.3 队列与重试

`商户端/src/utils/bluetooth-printer.ts` `BluetoothPrinter`：
- FIFO 打印队列
- 失败自动重试 3 次（指数退避：2s / 4s / 8s）
- 状态机：`disconnected → connecting → connected → printing → error`

### 7.4 多机并发（S6 占位 / P9 完整实现）

当前版本：单机串行  
**P9 扩展**：`PrinterPool` 维护多个 `BluetoothPrinter` 实例，按店铺路由

## 九、常见 Q&A

**Q1**：可以连接 USB 打印机吗？  
A：不支持。商户端仅支持蓝牙 4.0+ 的 BLE 打印机。USB 打印需用电脑端管理后台。

**Q2**：iOS 14+ 系统弹出蓝牙权限提示后拒绝了，怎么办？  
A：iOS 设置 → O2O → 蓝牙 → 开启

**Q3**：换手机后还需要重新配对吗？  
A：是的，蓝牙绑定与设备相关。新手机重走「扫描 + 连接」流程即可。

**Q4**：能不能远程指定打印？  
A：不支持。蓝牙打印必须 APP 与打印机都在线 + 蓝牙连接。
