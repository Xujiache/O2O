# ACCEPTANCE_P3_后端基础服务

## 一、功能验收

| 编号 | 场景 | 通过标准 |
|---|---|---|
| V3.1 | 小程序登录 | 提供 code → 返回合法 JWT；查库或新建用户 |
| V3.2 | 手机绑定 | 发送短信 → 5min 内可绑定；错验不通过 |
| V3.3 | 商户/骑手账密登录 | bcrypt 校验通过返回 token；错 5 次锁 30min |
| V3.4 | 管理员登录 | 返回 token + 菜单树 + 权限码列表 |
| V3.5 | Refresh Token | 过期 access 可由 refresh 续期；refresh 过期拒绝 |
| V3.6 | 权限守卫 | 缺权限返回 403 + 错误码 20003 |
| V3.7 | 用户 CRUD | 管理端分页查询、编辑、封禁/解封 |
| V3.8 | 商户入驻审核 | 提交 → 待审核；审核通过 → 可登录 |
| V3.9 | 骑手入驻 + 保证金 | 资质提交 → 审核 → 保证金缴纳记录 |
| V3.10 | 地址 CRUD | 默认切换唯一；软删除 |
| V3.11 | 黑名单 | 封禁后登录被拒；解封恢复 |
| V3.12 | 消息模板 CRUD | 按 code+channel 唯一 |
| V3.13 | 微信订阅消息 | 模拟触发 → push_record 成功；失败自动重试 3 次 |
| V3.14 | APP 推送 | JPush 通过 alias 推送到 target；mock 模式日志输出 |
| V3.15 | 短信 | 发送成功 → 记录；同号 60s 内拒发 |
| V3.16 | 站内信 | 写库 + 未读数；`mark-read` 可用 |
| V3.17 | 文件代理上传 | 返回公网 URL + file_meta id |
| V3.18 | 文件 STS 直传 | 前端可用临时凭证上传 MinIO |
| V3.19 | 文件签名 URL | 私有 bucket 通过签名可访问；直连 403 |
| V3.20 | 文件水印 | 商品图可选添加右下角水印 |
| V3.21 | 地理编码 | 地址解析为坐标，命中缓存 |
| V3.22 | 距离/路径 | 两点距离、骑行路径返回 |
| V3.23 | 配送范围校验 | 点在 polygon 内 → true |
| V3.24 | 骑手位置上报 | 10s 上报一次；Redis GEO 可 GEORADIUS；轨迹 TimescaleDB 可查 |
| V3.25 | 轨迹查询 | 指定 orderNo 可得完整 path（GeoJSON） |

## 二、非功能验收
| 项 | 标准 |
|---|---|
| 核心接口 P95 | ≤ 200ms |
| 鉴权接口 P95 | ≤ 150ms |
| 骑手上报接口 QPS | ≥ 500 单实例 |
| 单测覆盖 | ≥ 70% |
| Swagger | 接口、错误码、示例齐全 |
| 日志 | 每请求含 traceId，敏感字段不入日志 |
| 文档 | `docs/P3_后端基础服务/api/*.md` 齐全 |

## 三、安全验收
| 项 | 标准 |
|---|---|
| 密码 | bcrypt，cost ≥ 10 |
| Token | JWT HS512，`ver` 字段支持吊销 |
| 签名头 | 管理后台 X-Sign 校验通过 |
| 防重放 | 同一 nonce 60s 内拒绝 |
| 敏感日志 | mobile/id_card 仅显示 tail4 |
| 文件 | MIME + 大小 + 扩展名校验通过 |
