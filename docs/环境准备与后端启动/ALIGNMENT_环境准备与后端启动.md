### 问题 1: MySQL 的“重置”方式确认（关键决策）
目前的本地 MySQL 8.0 是通过 Windows 安装包安装的，并作为 `MySQL80` 服务运行。
- **方案 A (推荐 - 快速且安全)**: 停止 `MySQL80` 服务，清空现有的数据库数据目录 (`C:\ProgramData\MySQL\MySQL Server 8.0\Data`)，并使用 `mysqld --initialize-insecure` 重新初始化一个全新的空数据库实例，最后将 root 密码修改为 `Mm8822775`。这**在效果上等同于重装**，但速度极快且不依赖安装包。
